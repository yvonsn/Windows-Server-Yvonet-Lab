# Conclusiones

## Objetivo alcanzado

El laboratorio **Windows Server YVONET** permitió implementar una infraestructura básica de red utilizando Windows Server y un equipo cliente Windows 11 virtualizados mediante VirtualBox.

Durante el desarrollo del proyecto se configuró un dominio de Active Directory y se administraron usuarios, recursos compartidos y permisos NTFS, simulando un entorno empresarial.

---

## Conocimientos aplicados

Durante el proyecto se trabajó con las siguientes tecnologías y herramientas:

- VirtualBox
- Windows Server
- Windows 11
- Active Directory Domain Services (AD DS)
- DNS
- Unidades Organizativas (OU)
- Administración de usuarios
- Recursos compartidos
- Permisos NTFS
- Administración de redes básicas

---

## Competencias desarrolladas

Este laboratorio permitió desarrollar competencias relacionadas con:

- Instalación y configuración de servidores Windows.
- Administración de Active Directory.
- Organización de usuarios mediante OU.
- Gestión de permisos y recursos compartidos.
- Diagnóstico y resolución de incidencias.
- Documentación técnica de procedimientos.

---

## Aprendizajes

Uno de los aspectos más importantes del laboratorio fue comprobar que la administración de sistemas no consiste únicamente en realizar configuraciones, sino también en identificar problemas, analizar su origen y aplicar soluciones de forma ordenada.

La documentación realizada durante este proyecto servirá como referencia para futuras implementaciones y como evidencia práctica de las competencias adquiridas.

---

## Posibles mejoras futuras

Este laboratorio puede ampliarse incorporando nuevos servicios, como por ejemplo:

- Directivas de Grupo (GPO).
- DHCP.
- Servidor de impresión.
- PowerShell para automatización.
- Copias de seguridad.
- Monitorización de servidores.

---

## Conclusión

El proyecto **Windows-Server-YVONET-Lab** representa una implantación completa de un entorno Windows Server básico, documentada paso a paso y orientada a la adquisición de conocimientos prácticos en administración de sistemas y soporte IT.
