# Introducción al laboratorio YVONET

## Contexto del proyecto

YVONET es una empresa ficticia creada para simular un entorno empresarial de administración de sistemas.

El objetivo de este laboratorio es diseñar y documentar una infraestructura IT basada en Windows Server, reproduciendo situaciones habituales que pueden aparecer en un departamento de soporte técnico y sistemas.

---

## Objetivo del laboratorio

Este proyecto tiene como finalidad practicar tareas relacionadas con la administración de sistemas, entre ellas:

- Instalación y configuración inicial de un servidor.
- Creación de un dominio Active Directory.
- Administración de usuarios y grupos.
- Gestión de permisos y accesos.
- Configuración de recursos compartidos.
- Resolución y documentación de incidencias.

---

## Escenario empresarial

La organización YVONET cuenta con diferentes departamentos que necesitan trabajar con información compartida, manteniendo un control adecuado sobre los accesos.

Cada departamento tendrá usuarios, grupos y permisos definidos según sus necesidades.

---

## Alcance del proyecto

Durante el desarrollo del laboratorio se documentarán:

- Configuración del servidor Windows Server 2024.
- Creación del dominio YVONET.LOCAL.
- Configuración de Active Directory.
- Creación de usuarios y grupos.
- Administración de permisos NTFS.
- Acceso a carpetas compartidas desde equipos cliente.
- Resolución de incidencias.

---

## Entorno utilizado

**Servidor:**
- Sistema operativo: Windows Server 2024
- Nombre: SVR-YVONET
- Dominio: YVONET.LOCAL

**Cliente:**
- Sistema operativo: Windows 11

---

## Objetivo profesional

Este laboratorio está orientado a reforzar competencias para puestos de:

- Soporte IT / HelpDesk L1.
- Administración básica de sistemas.
- Resolución de incidencias.
- Documentación técnica.

El proyecto se desarrolla siguiendo buenas prácticas de documentación y organización utilizadas en entornos profesionales.
