# 03. Instalación y configuración de sistemas

## Introducción

En esta fase del laboratorio **YVONET** se realiza la instalación y configuración inicial de los sistemas operativos que forman parte de la infraestructura virtual.

El entorno está compuesto por dos máquinas virtuales creadas mediante **Oracle VirtualBox**:

- Un servidor con **Windows Server 2022**, que actuará como servidor principal de la infraestructura.
- Un equipo cliente con **Windows 11**, que será utilizado para conectarse al servidor y trabajar dentro del entorno de dominio.

Durante esta fase se realizan las siguientes tareas:

- Instalación de Windows Server 2022.
- Instalación de Windows 11.
- Configuración inicial de ambos sistemas.
- Identificación de los equipos dentro del laboratorio.
- Preparación de los sistemas para la configuración de red y servicios posteriores.

Una vez finalizada esta etapa, el servidor y el cliente quedan preparados para la configuración de red y la implementación de los servicios de infraestructura de **YVONET**.
