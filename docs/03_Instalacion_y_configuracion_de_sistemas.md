# 03. Instalación y configuración de sistemas

## Introducción

En esta fase del laboratorio **YVONET** se realiza la instalación y configuración inicial de los sistemas operativos que forman parte de la infraestructura.

El entorno está compuesto por un servidor con **Windows Server 2022** y un equipo cliente con **Windows 11**, previamente preparados en la fase anterior.

Durante esta etapa se realizan las siguientes tareas:

- Instalación de Windows Server 2022.
- Instalación de Windows 11.
- Configuración inicial de ambos sistemas operativos.
- Identificación de los equipos dentro del laboratorio.
- Preparación de los sistemas para las fases posteriores de configuración.

Una vez finalizada esta fase, el servidor y el cliente quedan operativos y preparados para realizar la configuración de red y la implementación de los servicios de infraestructura de **YVONET**.
