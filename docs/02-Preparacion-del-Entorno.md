# Preparación del entorno

## Virtualización

Para la creación del laboratorio YVONET se utilizó VirtualBox como plataforma de virtualización.

La virtualización permite simular una infraestructura empresarial utilizando máquinas virtuales independientes dentro de un equipo físico.

## Máquina virtual del servidor

Se creó una máquina virtual destinada a alojar el controlador de dominio.

Configuración inicial:

- Sistema operativo: Windows Server 2024
- Nombre del equipo: SVR-YVONET
- Función: Servidor principal del dominio

## Red

La máquina virtual fue configurada para permitir la comunicación entre el servidor y los equipos cliente del laboratorio.
